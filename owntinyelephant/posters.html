Python Concepts for Beginners

A simple yet effective poster for aspiring and proficient computer scientists and developers of all age rages to improve and refresh their knowlege on Python concepts. The content of this poster is curated for beginners who are new to coding, however programmers of all calibres will benefit from the content of this poster. This poster walks the user through all aspects of Python coding, from data types to loops and conditionals, giving colour coded examples all the way through. This poster is printed on both sides, with the intention to be used as a flashcard set, letting learners cover the contents of the poster and practice active recall to quickly develop their mastery of Python programming.
